# PRD: AI 협업 개인 브랜딩 랜딩 페이지

**문서 버전**: v1.0  
**작성일**: 2026-04-03  
**문서 유형**: Product Requirements Document

---

## 1. 제품 개요

### 1.1 제품명
AI 협업 개인 브랜딩 랜딩 페이지

### 1.2 목적
AI 활용이 막막한 직장인에게 실무에서 바로 적용 가능한 AI 협업 방법을 소개하고, 업무 생산성 향상과 창의적인 일에 집중하는 즐거움을 제공하는 개인 브랜딩 페이지.

### 1.3 핵심 가치 제안
> "막막함은 줄이고, 일은 더 빠르고 똑똑하게. AI와 함께 더 가치 있는 일에 집중하는 사람이 되세요."

---

## 2. 타겟 사용자

### 2.1 주요 타겟
- **직군**: 일반 직장인 (비개발직군 포함)
- **페인포인트**: AI 활용에 막막함을 느끼거나 어디서부터 시작해야 할지 모르는 분
- **니즈**: 이론보다 실무에 즉시 적용 가능한 실전 노하우

### 2.2 사용자 여정
1. 랜딩 페이지 방문
2. 콘텐츠 주제 확인 및 공감
3. CTA 버튼(구글 폼) 클릭 → 신청/구독 전환

---

## 3. 기능 요구사항

### 3.1 페이지 구성 섹션

| 섹션 | 내용 | 우선순위 |
|------|------|----------|
| Hero | 핵심 가치 제안 문구 + CTA 버튼 | P0 |
| 콘텐츠 주제 카드 | 3개 주제 소개 | P0 |
| CTA (하단) | 구글 폼 연결 버튼 | P0 |
| 네비게이션 | 로고/브랜드명 + 앵커 메뉴 | P1 |
| 푸터 | 저작권 표기 | P1 |

### 3.2 콘텐츠 상세

**Hero 섹션**
- 메인 헤드라인: AI 협업 핵심 메시지
- 서브 카피: 타겟 페인포인트 공감 문구
- CTA 버튼: "지금 시작하기" (구글 폼 링크 연결)

**콘텐츠 주제 카드 (3개)**

| # | 주제 | 핵심 메시지 |
|---|------|-------------|
| 1 | AI로 일하는 방식이 어떻게 달라지는가 | 막막함은 줄이고, 일은 더 빠르고 똑똑하게 |
| 2 | 직장인을 위한 실무형 AI 협업법 | 이론이 아니라 바로 써먹는 실전 활용 |
| 3 | AI와 함께 더 창의적으로 일하는 사람의 브랜드 | 생산성 향상을 넘어, 더 가치 있는 일에 집중하는 사람 |

**CTA 섹션**
- 강조 문구: 행동 유도 메시지
- 버튼: 구글 폼 URL 연결 (추후 업데이트)

---

## 4. 비기능 요구사항

### 4.1 기술 스택
- HTML5, CSS3, Vanilla JavaScript
- 외부 라이브러리 최소화 (Google Fonts 허용)
- 단일 HTML 파일로 배포 가능

### 4.2 디자인 가이드라인
- **레퍼런스**: Toss 사이트 스타일
- **톤**: 밝고 신뢰감 있는 라이트 테마
- **디바이스**: 데스크탑 우선 (Desktop-first)
- **접근성**: WCAG 2.1 AA 수준 권고

### 4.3 성능
- 초기 로딩: 3초 이내 (LCP 기준)
- 외부 폰트 외 CDN 의존 최소화

## 5. CTA 버튼 스펙

| 항목 | 내용 |
|------|------|
| 버튼 텍스트 | "지금 신청하기" 또는 "무료로 시작하기" |
| 링크 | 추후 구글 폼 URL 연결 (`#` 임시 처리) |
| 위치 | Hero 섹션 + 하단 CTA 섹션 (총 2곳) |
| 스타일 | Primary 버튼 (브랜드 컬러) |

## 6. 마일스톤

## 7. 성공 지표 (KPI)

| 지표 | 목표 |
|------|------|
| 구글 폼 클릭률 (CTR) | 랜딩 페이지 방문자의 10% 이상 |
